# 第 23 章计划：晚自习座位表

## 章节目标

- 换场域，从办公室资料转到三班内部、晚自习、班主任/数学老师压力。
- 让第 22 章的句号创可贴产生余波：林知夏没有公开回应，但心里开始松动。
- 进一步压实“受欢迎的人最难藏住喜欢”：同学目光、班主任提醒、成绩波动同时出现。

## 主要事件

1. 周五下午，三班班主任宣布晚自习座位微调，理由是最近班里浮躁、临近期中后第一次阶段测。
2. 林知夏因手上新创可贴被叶蔓发现，叶蔓猜到不是医务室随便拿的，但没有逼问。
3. 数学老师发小测，林知夏又在一道基础函数题上丢分，被点名去办公室门口谈话。
4. 班主任顺势提到她最近“帮英语老师的事很多、走廊上被人议论也多”，让她把心收回来。
5. 晚自习座位调整后，林知夏被调到靠讲台的位置，叶蔓坐远；她第一次明显感到孤立。
6. 电影配音试听名单要确认，陈既白托周应川送来七班最终名单，名单最后一格仍空着，旁边夹着旧音箱转接头的借用条。
7. 林知夏把三班名单交给班主任签字时，班主任看见七班借用条，问“怎么又是七班？”
8. 结尾：林知夏晚上回家打开手机，看见句号发来一条很短的短信：“明天试听，我不在名单里。”她盯着那张创可贴，没有马上回。

## 情绪走向

- 林知夏：被班主任和同学目光夹住，想把一切退回正轨，却因为创可贴和短信退不干净。
- 陈既白：不直接出现太多，以“空格名单、借用条、短信”保持存在感。
- 叶蔓：保护但不追问，给林知夏留一口气。

## 注意事项

- 班主任不是反派，语气现实、克制。
- 不要写成抓早恋现场，重点是“状态下滑”和“被议论”的压力。
- 单章要有清晰事件链：小测失误 -> 谈话 -> 调座位 -> 名单/借用条引发追问 -> 短信钩子。
- 结尾保持酸涩，不和好。
